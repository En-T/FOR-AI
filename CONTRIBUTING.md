# Как участвовать в проекте

## Требования перед разработкой

1. Установить Python 3.9+
2. Установить зависимости: `pip install -r requirements.txt`
3. Запустить миграции: `python manage.py migrate`

## Рабочий процесс

1. Создать ветку для вашей фичи: `git checkout -b feature/your-feature`
2. Сделать изменения
3. Сделать коммит: `git commit -am 'Add some feature'`
4. Загрузить в репозиторий: `git push origin feature/your-feature`
5. Создать Pull Request

## Стиль кода

- Следовать PEP 8
- Использовать понятные имена переменных
- Добавлять комментарии для сложной логики
- Писать docstring для функций и классов

## Нужна помощь?

Создайте Issue на GitHub с описанием проблемы.
