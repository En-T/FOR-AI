# Морской бой (Pygame)

Запуск:

```bash
pip install -r requirements.txt
python main.py
```

Управление:
- **Меню**: выбор сложности (Легко/Средне), новая игра, выход.
- **Расстановка**:
  - ЛКМ — поставить корабль.
  - **R** — повернуть (горизонтально/вертикально).
  - **Backspace** — удалить последний корабль.
  - **Space** — автозаполнение (опционально).
- **Игра**: ЛКМ по правому полю для выстрела.
